Project Management System

Our Project Management System is a web based application created to be used by corporate businesses for managing company projects. The system is built for project management, offering features such as: project creation, project overview, project analytics and calendar views for multiple projects. Other services offered by the project management system include employee profiles which track analytics on past projects, an internal web based notification system which co-aligns with an email notification service and a bare-bones particle swarm optimization algorithm for allocating optimal teams for new projects.
