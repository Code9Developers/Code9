Project Management System

A web based application created to be used by corporate businesses for managing company projects, employees & optimised team allocations

Live Demo* Quick Start*

Built For Project Management

We offer a wide range of tools to the project manager so that they can handle their team with ease. Making the over all user experience of managing your team and projects less complicated.

Tools

Tools such as:

  • project overview, creation & analytics
  • calendar views for multiple projects
  • AI enhanced team allocation
  • employee analytics tracking
  • internal notifications system

Technologies

In our project we used common web development tools as well as the Mean stack.

Mongodb

A noSQL database that uses JSON objects for efficient transportation of data.

Nodejs

A lightweight and efficient JavaScript runtime that runs on Chromes V8 engine. It uses a large open-source library of packages.

HTML 5

The latest version of HTML that allows us to build powerful websites with newer tags and integrations.

Express

Express is a minimal and flexible Node.js web application framework that provides a robust set of features for web and mobile applications.

CSS 3

A styling language used to bring life to HTML5.

JS

A high-level interpreted programming language used to make web page interactive. This along with HTML and CSS form the core modules of web development.
